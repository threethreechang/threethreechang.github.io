1.1 工具

  • Node.js

    新版nodejs集成了npm

  • git

  • hexo:

    npm install hexo-cli -g

查看版本，确认安装成功。

1.2 搭建仓库

在github中：

  • new repository

    建立名为“username.github.io”的仓库

    比如我github名叫threethreechang 则建立threethreechang.github.io

  • public

  • add a README file

1.3 配置ssh key

  1. 任意文件夹下Git Bash Here

  2. ssh

  3. ssh-keygen -t rsa -C "邮件地址"

    初次生成sshkey 需敲4次回车，若已存在会提示existed

    生成id_rsa.pub文件 位于/c/Users/YOURNAME/.ssh/id_rsa

  4. 单击个人->settings->SSH and GPG keys->new SSH key

  5. 起个Title名，将id_rsa.pub中内容复制到Key下并确定

  6. 测试ssh绑定是否成功

    ssh -T git@github.com #提示 hi username 即成功

1.4 本地搭建博客

  1. 新建空文件夹作为博客项目文件夹 并git bash here
  2. hexo init #拷贝hexo框架至本地
  3. hexo s #静态生成本地hexo页面

1.5.部署至网络

  1. 打开hexo框架中的_config.yml

  2. 文件末尾添加内容并保存：

  3. npm install hexo-deployer-git --save

  4. hexo g #hexo generate 生成静态文件

  5. hexo d #hexo deploy 部署网站

  6. 弹出open SSH窗口

    -输入用户名

    -输入令牌序列号

    令牌获取：个人->settings->Peronal access tokens->新建，配置期限和权限->得到了序列号

    第一次生成的时候能看见令牌 之后令牌会被隐藏。截图留用
